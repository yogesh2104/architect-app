# Landing Page AI Image Prompts

These prompts are based on the current landing page structure in this project: premium architecture + interior studio, modern luxury style, clean white/slate background, warm orange accent, elegant furniture, calm lighting, high-end craftsmanship.

Use these prompts in Gemini, Qwen, or any image generator. For best results, keep the same color language across all sections:

- Primary mood: premium, calm, refined, trustworthy
- Color palette: warm white, soft beige, slate gray, natural wood, muted charcoal, subtle orange accents
- Style: photorealistic, editorial interior photography, luxury architecture branding, natural daylight, high detail
- Avoid: clutter, oversaturated colors, cartoon style, unrealistic furniture proportions, noisy backgrounds, distorted hands, extra objects

## 10. Strong Universal Negative Prompt

Use this as an extra negative prompt if the tool supports it:

```text
low quality, blurry, cartoon, illustration, overexposed, oversaturated, distorted furniture, bad anatomy, duplicate objects, cluttered room, messy composition, warped walls, unrealistic lighting, text, watermark, logo, frame, low detail
```

## 11. Best Practice For Consistent Results

- Keep all images in one visual family: same warm neutral palette and premium editorial realism.
- For hero and section banners, ask for "negative space for text overlay".
- For product cards, ask for "centered subject" and "clean background".
- For trust sections, avoid overly dramatic scenes; keep them calm and believable.
- If Gemini or Qwen gives too much decoration, add: `minimal decor, uncluttered luxury interior, clean composition`.
- If faces look weak, avoid human portraits and use lifestyle interiors instead.
